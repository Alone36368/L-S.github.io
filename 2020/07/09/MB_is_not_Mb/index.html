MB/s != Mb/s

发表于 2020-07-09 更新于 2022-07-11 分类于 惊

MB/s != Mb/s

今天居然发现这个神奇的事情，最开始是认为电脑网速和手机不一样（可能屏幕大，下载速度小），今天发现电脑下载速度和手机的速度好像有点等量关系关系（差不多两倍）。然后网上查查得到的结果是MB=MByte（兆字节）,而Mb=Mbit（兆比特）。正如我们所知1 Byte = 8 bit.一般数据传输用bit，而磁盘储存（即下载时）用Byte。

意思就是如果电脑显示的网速为3MB/s，那任务管理器应该大概显示的是24Mbps=24Mbit


所以通常，运营商说的1M宽带的M是指Mb/s，也就是Mbps，换算一下的话，1M宽带下载速度也就是125KB/s，再去掉损耗的话就是120KB/s左右。以此类推，10M宽带的最快下载速度是1.25MB/s，100M的宽带最快下载速度是12.5MB/s。

说了个寂寞

所以手机开热点为什么显示却是两倍左右的7Mb/s ？？？

手机开热点相当于充当路由器，电脑连接上热点，那么手机其实是两边一起传输的，即下载数据和上传到电脑，但是上传到电脑是无需流量的，但是会计算到总流量中，所以显示也会是两倍左右。
